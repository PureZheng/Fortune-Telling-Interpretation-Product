# 合规与风控规范（MVP）

## 1. 产品定位声明

- 服务定位：娱乐与参考，不构成专业建议。
- 展示位置：首次进入、结果页底部、设置页。

建议文案：

“本服务提供的内容仅供娱乐与参考，不构成医疗、法律、投资等专业意见，请理性看待。”

## 2. 敏感内容拦截

### 2.1 拦截范围

- 生死预测、自残、违法行为、重大疾病判断。
- 明确索要医疗诊断、投资确定收益、法律结论。

### 2.2 处理策略

- 命中高危词：拒答并引导寻求专业帮助。
- 命中中危词：弱化输出，提供一般性建议。
- 所有命中事件记录审计日志（不记录完整隐私文本）。

## 3. 未成年人保护

- 注册或首次使用时做年龄确认。
- 对未成年人展示额外提醒，限制深度问题频率。

## 4. 隐私与数据治理

- 最小化采集：仅保留生成所需字段。
- 用户可在设置页申请删除已保存的个人资料与相关数据。
- 删除请求处理时限：7 天内完成。

## 5. 输出安全约束

- 使用非确定性表达，不输出宿命论指令。
- 不使用恐吓、威胁、绝对否定类措辞。
- 默认带有 `riskNote` 字段提示理性决策。
