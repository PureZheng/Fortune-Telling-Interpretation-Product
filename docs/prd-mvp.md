# 算命产品 MVP PRD

## 1. 产品目标

面向普通用户提供轻量、可复访的玄学解读体验，验证用户是否愿意持续使用“每日运势 + 问题解读”服务。

## 2. 用户画像

- 对命理、占卜、运势内容感兴趣的泛娱乐用户。
- 主要诉求是“快速获得解释和建议”，而非专业研究。
- 使用场景集中在早晚通勤、睡前、遇到决策困惑时。

## 3. 核心用户路径

1. 用户进入首页，选择“开始解读”。
2. 填写基础信息（生日、可选出生时辰、问题方向）。
3. 获取基础命理解读结果（结构化 + 自然语言）。
4. 当天再次访问时查看“每日运势”卡片。
5. 有具体问题时进入“算卦”，输入问题并获得结果。

## 4. MVP 功能清单

### 4.1 信息录入

- 必填：出生日期（公历）。
- 选填：出生时间、性别、出生地、问题方向。
- 校验：日期合法性、时区默认值、字段长度限制。

### 4.2 基础命理解读

- 输出字段：`summary`、`dimensions`、`advice`、`riskNote`、`confidence`。
- `dimensions` 默认包含：事业、感情、财运、健康。
- 内容形式：短摘要 + 分项解释 + 可执行建议。

### 4.3 每日运势

- 生成频率：每用户每日一次（按时区）。
- 输出内容：总运分、四维分项、今日建议、注意事项。
- 展示形式：卡片 + 标签（顺势/平稳/谨慎）。

### 4.4 算卦解读

- 输入：用户问题（必填，长度限制）。
- 起卦方式：随机卦象（MVP 阶段），后续可扩展交互式起卦。
- 输出：卦象、卦辞摘要、现代解释、行动建议、风险提示。

## 5. 非功能需求

- 结果生成时延：首屏目标 30 秒内。
- 可用性：接口成功率目标 >= 99%（MVP 阶段）。
- 可观测性：记录请求耗时、模型失败率、敏感拦截率。

## 6. 不在 MVP 范围

- 支付系统与会员体系。
- 社区互动、分享裂变、积分体系。
- 专业级命盘可视化与多流派并行计算。
