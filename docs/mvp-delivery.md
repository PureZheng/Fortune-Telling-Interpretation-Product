# MVP 交付计划与验收

## M1（1-2 天）

- 完成 PRD、API 规范、合规文档。
- 完成目录结构和规则/提示词模板。
- 验收：文档可指导开发，无关键字段缺失。

## M2（3-5 天）

- 实现核心接口（解读、运势、算卦、追问助手、用户资料等）。
- 跑通规则层 -> 模型层 -> 结果存储链路。
- 验收：解读、每日运势、算卦接口均可返回结构化数据。

## M3（2-3 天）

- 增加敏感拦截、免责声明、审计日志。
- 验收：高危输入可拦截，日志可追踪。

## M4（1-2 天）

- 小流量灰度发布。
- 观察 D1 留存、日运打开率、拦截率与错误率。
- 验收：系统稳定，核心指标可监控。

## 首月指标

- D1 留存率
- 每用户周均解读次数
- 每日运势打开率
- 算卦功能使用率
- 免责声明确认率
- 敏感投诉率
